# Video Metadata Dataset Documentation

## Overview

This document provides comprehensive technical specifications for the video metadata dataset stored in the `data/` directory. The dataset has been partitioned from a monolithic JSON file (`full_data.json`) into multiple smaller files for improved manageability and version control compatibility.

## Dataset Characteristics

- **Source File**: `full_data.json` (483 MB)
- **Total Records**: 125,978 video metadata entries
- **Data Structure**: Array of JSON objects
- **Partitioning Strategy**: Fixed-size chunks of 100 records per file
- **Output Directory**: `data/`
- **File Naming Convention**: `videos_page_{page_number}.json`
- **Total Output Files**: 1,260 files (1,259 × 100 records + 1 × 78 records)
- **Serialization**: Pretty-printed JSON with 2-space indentation
- **Version Control**: Standard Git tracking (no Git LFS required)

## Record Schema

Each video metadata record is a JSON object with the following schema:

### Core Video Properties
| Field | Type | Description | Example |
|-------|------|-------------|---------|
| `embed` | string | HTML iframe embed code for Pornhub video player | `"<iframe src=\"https://www.pornhub.com/embed/...\" ...></iframe>"` |
| `title` | string | Video title/description | `"Gen Padova - Cum Bot Brunette Petite Girl Orgasms"` |
| `duration` | string | Video length in seconds (numeric string) | `"185"` |
| `views` | string | Total view count (numeric string) | `"2351874"` |
| `likes` | string | User likes count (numeric string) | `"3014"` |
| `dislikes` | string | User dislikes count (numeric string) | `"416"` |

### Visual Assets
| Field | Type | Description | Format |
|-------|------|-------------|--------|
| `thumbnail` | string | Primary thumbnail image URL | HTTPS URL to JPG |
| `thumbnail2` | string | Secondary thumbnail image URL | HTTPS URL to JPG |
| `screenshots` | string | Primary screenshot gallery | 16 semicolon-separated HTTPS URLs |
| `screenshots2` | string | Secondary screenshot gallery | 16 semicolon-separated HTTPS URLs |

### Content Classification
| Field | Type | Description | Format |
|-------|------|-------------|--------|
| `tags` | string | Content tags/keywords | Semicolon-separated values |
| `categories` | string | Official category classifications | Semicolon-separated values |
| `actors` | string | Featured performer names | Comma or semicolon-separated |
